MemoryAllocation Project Simulates memory allocation algorithms in a liked list It provides a command line interface: add n s - command adds job number n of size s to the list of jobs on the system jobs - prints the list of jobs on the system list - prints the current contents of the linked list segments ff n - invokes the First Fit allocation for job number n nf n - invokes the Next Fit allocation for job number n bf n - invokes the Best Fit allocation for job number n wf n - invokes the Worse Fit allocation for job number n de n - deallocates the specified job number n
